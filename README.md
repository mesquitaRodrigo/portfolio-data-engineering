# Portfolio Data Engineering

Projeto completo de Engenharia de Dados desenvolvido utilizando PostgreSQL, Python, Parquet, Docker e Metabase.

## Arquitetura

ERP PostgreSQL
↓
Raw Layer (Parquet)
↓
Curated Layer (Star Schema)
↓
Data Warehouse PostgreSQL
↓
Analytics Layer
↓
Metabase Dashboard

## Tecnologias

* Python
* PostgreSQL
* SQLAlchemy
* Pandas
* Parquet
* Docker
* Metabase

## Funcionalidades

* Extração automatizada
* Transformação de dados
* Modelagem dimensional
* Data Warehouse
* Data Quality
* Testes automatizados
* Dashboards executivos
